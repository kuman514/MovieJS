# MovieJS
VanillaJS를 이용하여 만든, 웹 동영상 플레이어 앱입니다.   
[앱 사용해보기](https://kuman514.github.io/MovieJS/index.html)

# Features (구현 요소)
- HTML5의 `<video>` 태그를 이용한 비디오 재생 컨트롤
- 동영상 파일을 불러오는 즉시 자동으로 재생
- 반복 재생 여부 토글 기능
- CSS의 `@media` 쿼리를 이용한 최소 해상도 지정
- `.avi`, `.mp4`, `.mkv` 파일 지원
